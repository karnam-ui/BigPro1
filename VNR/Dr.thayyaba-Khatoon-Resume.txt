PROJECT DONE: (PhD)
Title A Novel Approach for search query performance
improvement on Biomedical Literature
Environment Used Windows, Linux, java, Python.
Duration 3 year
Description A methodology is proposed to enhance aforementioned
techniques involved in query processing. Various pre-
processing techniques and term weighting functions are
explored with possible enhancements. Similarly
enhancements are proposed for similarity measures and
retrieval functions. An algorithm known as Multi-Kernel
Fuzzy C Means (MKFCM) is proposed to have topic
based clustering to drive performance of query processing
further. Besides, the research focused on the query
expansion methods based in different relevance feedback
approaches. Query Expansion (QE) is a process of
reformulating user query to improve query understanding
and retrieval performance in Information Retrieval (IR).
It also threw light into the problem of over penalizing
longer length documents in predecessors by proposing a
solution that strikes balance between document length
normalization and giving importance to longer length
documents. Complete research and findings are presented
in different chapters. The following sections provide
more insights on the research carried out in this thesis.
