The library was established in 1995 along with the Institution with a
          collection of 500+ books. Now the library has a collection of 82,931
          volumes and 17,585 titles. The library has well maintained reference
          collection that includes 15,188 books, Encyclopedias, Handbooks and
          Directories, with quick reference facility. The library is well
          equipped with various learning resources enabling the stakeholders to
          access in physical and/or online mode. The library has an annual
          subscription to 1280 online e-journals and 323 print journals. The
          library is a member of DELNET, N-LIST, NDL (National Digital Library)
          which helps to provide hassle-free e-learning unconditionally. The
          books are shelved as per the Dewey Decimal Classification scheme to
          facilitate easy location of the books by following Open Access System.
Library Automation:The library is automated
                with Koha Software with RFID technology. The College Library
                catalogue can be accessed through the OPAC (Online Public Access
                Catalogue) which provides information of the library collection
                online. The library OPAC is accessible with the URLhttp://vnr-vjietopac.2cqrkoha.com/The library has subscription to e-Books of packt , Lakshmi
                publications, Videeya Books, and N-LIST. The library also
                encourage students to access free content from AICTE sponsored
                Ekumbh website.  The library also maintains a Book Bank for
                SC/ST students which provides an opportunity to own additional
                books offered by the Social Welfare Department, Govt. of
                Telangana. The library maintains an EPS Book bank to issue
                additional books for the economically weak students those
                parental annual income is less than Rs. 2 lakhs. ;.`
The digital library consists of 20 computers connected to an
                  internet facility. It is open from 8 am to 8 pm along with the
                  Central Library. Here, the students access the subscribed
                  online content, NPTEL video lectures, and can practice online
                  mock tests, etc.
The above-mentioned e- journal packages are accessible through
                  e -library KNIMBUS software and m-library mobile app also.
The library provides remote access to its subscribed content
                  through KNIMBUS platform. It can be reached through the URL:https://vnrvjiet.knimbus.comThis facility can be used by all the staff and students by
                  self-registration with institute mail ID and with self created
                  password. Here the users can access databases, e-Textbooks, e-
                  journals, previous question papers, lecture notes, open access
                  content provided by staff. This knimbus e-library can be
                  accessed through smart phones by downloading mobile app
                  mLibrary from Google Play Store.
The library encompasses a Reading room and a Reference section
                  with reprographic and scanning facilities. The library also
                  having the following sections to cater the needs of users.
The library has subscription to six English and two Telugu
                  daily newspapers which helps to stakeholders to be up to date
                  on current affairs.
The library has established the Career Vision Promotion Centre
                  (CVPC) with a computer and internet facility, LCD projector,
                  25 seating capacity for watching self-learning videos, TED
                  videos, NPTEL Lectures by the users on subject domains of
                  their interest.
The library is opened from 8.00 a.m. to 8.00 p.m. on all working
                days and 9.00 a.m. to 4.00 p.m. during semester end
                examinations. The bus facility is provided up to 8.00 p.m. to
                utilize sports and library facilities.
Here the library user can find the details of a particular book
                availability by usinghttp://vnr-vjietopac.2cqrkoha.com/
The library provides remote access facility to its purchased
              content like IEEE, Science Direct, SpringerLink, ASME, ASCE and
              other uploaded content by the Library. This library is designed by
              using federated search engine which can search entire purchased
              content with help of key words. An user can search the required
              content with set of key words instead of registering and searching
              each database separately. This means the user will get search
              results from ASME, ASCE, IEEE, SpringerLink, etc. at single
              search. The search results can be filtered by using year,
              publications, type of publication etc. All these facilities can be
              accessed athttps://vnrvjiet.knimbus.comby self-registration. The training will be provided at digital
              library to utilize this e-library.

              The final year projects will be checked at department level by
              consulting the department coordinator. Each department is
              allocated with 25 user licenses to conduct the plagiarism of
              publications written by their students and staff. Those who has
              written research papers, M.Tech thesis can approach library for
              plagiarism check, if their guide is not having Turnitin license.
The Grammarly software is available at digital library for
